Subject Overview – MCA 2nd Semester

This semester includes the following major subjects:

  • Object Oriented Programming (OOP) – Focuses on concepts like classes, objects, inheritance, polymorphism, abstraction, and encapsulation.
  • Operating Systems (OS) – Covers process management, memory management, file systems, scheduling algorithms, and synchronization.
  • Database Management Systems (DBMS) – Includes relational models, SQL, normalization, transactions, indexing, and database design.
  • Data Structures & Analysis of Algorithms (DSA) – Covers stacks, queues, linked lists, trees, graphs, sorting, searching, and algorithm efficiency.

Students are advised to revise each unit thoroughly before attempting previous year papers. Attempt the papers in a timed environment to simulate real examination conditions.
